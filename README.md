# コレ｜理工系講義・教育支援サイト（見やすさ改善版）

大学・高専等からの講義依頼を受けるための、GitHub Pages向け静的サイトです。
外部ライブラリを使わず、`index.html` と `styles.css` だけで動作します。

## 改善した点

- 最初の画面で「講義依頼を受け付けていること」が分かる構成
- 担当実績、対応科目、講義の特長を明確に分離
- 長い説明を減らし、視線を縦に流しやすいレイアウトへ変更
- 講義テーマを一覧で比較しやすい形式に変更
- スマートフォン用メニューを追加
- キーボード操作や文字サイズにも配慮

## 公開前に変更する箇所

`index.html` 内の次の文字列を、実際のメールアドレスに一括置換してください。

```text
YOUR_EMAIL@example.com
```

必要に応じて、氏名「コレ」、専門分野、教育実績、講義例も修正してください。

## GitHub Pagesで公開する手順

1. GitHubで新しいリポジトリを作成します。
2. `index.html` と `styles.css` をリポジトリ直下へアップロードします。
3. リポジトリの `Settings` を開きます。
4. 左側の `Pages` を選びます。
5. `Build and deployment` の `Source` を `Deploy from a branch` にします。
6. Branchを `main`、フォルダを `/ (root)` にして保存します。
7. 公開URLが表示されたら、名刺のQRコードに設定します。
